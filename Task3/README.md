# Smart Factory Machine Monitoring System

## CodeAlpha IoT Internship

### Task 3

**Project:** Smart Factory Machine Monitoring System

**Platform:** Arduino Uno (Simulation)

**Software:** Tinkercad

### Objective
To monitor machine temperature using a TMP36 temperature sensor and alert the operator when the temperature exceeds a predefined limit.

### Components Used
- Arduino Uno
- TMP36 Temperature Sensor
- Red LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

### Working Principle
The TMP36 sensor measures the machine temperature continuously. Arduino reads the sensor value and displays the temperature on the Serial Monitor. If the temperature exceeds the preset limit, the warning LED turns ON to alert the operator.

### Applications
- Smart Factory
- Industrial Automation
- Machine Health Monitoring
- Predictive Maintenance
