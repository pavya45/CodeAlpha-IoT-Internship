# Motion Detection System

## CodeAlpha IoT Internship

### Task 2

**Project:** Motion Detection System

**Platform:** Arduino Uno (Simulation)

**Software:** Tinkercad

**Description:**
This project detects motion using a PIR Motion Sensor. When motion is detected, the LED turns ON. Otherwise, the LED remains OFF.

**Components Used**
- Arduino Uno
- PIR Motion Sensor
- LED
- 220Ω Resistor
- Jumper Wires
